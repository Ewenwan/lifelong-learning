【1】 ***Java编程思想*** 讲了很多通用编程实践方面的内容，刚学`Java`时看得是这本，对我的整体面向对象思想的影响也很大。
如果已经有较多的编程的经验（不限`Java`语言），会觉得内容有些罗嗦。

【2】 ***JAVA核心技术*** 我读的是06年版的，讲`Java 5`。需要补充一下`Java 7/8`，方便自己体系的使用新增的内容。
